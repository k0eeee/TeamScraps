# How to Run TeamScraps

## Requirements
- Java 17 or later
- IntelliJ IDEA or Maven installed

## Running the Game
1. Open the project in IntelliJ.
2. Ensure `src/main/java` is marked as Sources Root.
3. Ensure `resources` is marked as Resources Root.
3. Run the `Core` class inside `engine`.
4. Use arrow keys to control the snake.

## Running Unit Tests
1. Ensure `src/test/java` is marked as Test Root.
2. Right-click the `entity_test` package inside `src/test/java`.
3. Select “Run Tests”.
4. JUnit 5 will execute all test classes.

## Controls
- Arrow keys to move

## Note:
- Keep `target` marked as Excluded
